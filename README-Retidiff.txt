# RETI-Diff: Illumination Degradation Image Restoration

This module integrates the [RETI-Diff](https://github.com/ChunmingHe/Reti-Diff) model for advanced NIR image enhancement in our workflow.

## 1. Source

The RETI-Diff model and all associated code in this directory are based on the original open-source implementation:

> [https://github.com/ChunmingHe/Reti-Diff](https://github.com/ChunmingHe/Reti-Diff)

All credits for algorithmic innovation and core implementation belong to the original authors.

## 3. Usage

* Please refer to the original [RETI-Diff repository](https://github.com/ChunmingHe/Reti-Diff) for complete setup, pre-trained model download, and inference instructions.
* In this project, RETI-Diff is used for post-acquisition NIR image restoration and enhancement, especially in biomedical and night-vision imaging scenarios.
* If you use this module, ensure you also acknowledge the original authors in any derived work or publication.
